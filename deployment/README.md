# 企业私有知识库问答系统 - 前端部署

## 📦 部署脚本

本目录包含企业私有知识库问答系统的前端一键部署脚本，支持 Linux 和 macOS 系统。

## 🚀 快速开始

### 1. 环境要求

- Node.js >= v18.0.0
- npm >= v9.0.0

详细安装指南请查看：[部署指南.md](./部署指南.md)

### 2. 快速部署

```bash
# 进入部署目录
cd deployment

# 设置执行权限（首次使用）
chmod +x deploy.sh

# 环境检查
./deploy.sh --check

# 启动开发服务器
./deploy.sh --dev

# 或构建并启动生产服务器
./deploy.sh --prod
```

### 3. 访问应用

服务启动后访问：http://localhost:5173

## 📋 部署选项

| 选项 | 说明 |
|------|------|
| `--check` | 环境依赖检查 |
| `--build` | 构建项目 |
| `--dev` | 启动开发服务器 |
| `--prod` | 构建并启动生产服务器 |
| `--stop` | 停止服务 |
| `--clean` | 清理构建文件 |
| `--help` | 显示帮助 |

详细使用说明请查看：[部署指南.md](./部署指南.md)

## ⚙️ 配置

编辑 `config/app.conf` 文件修改配置：

```bash
APP_PORT=5173           # 应用端口
NODE_MIN_VERSION=18.0.0 # 最低 Node.js 版本
```
